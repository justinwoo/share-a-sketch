## Running / Development

### UI only

* `ember server`
* Visit your app at [http://localhost:4200](http://localhost:4200).

run tests with `ember test`

build with `ember build`

### The Real McCoy

* run `redis-server` locally
* `node .` (or `nodemon`)
* Open up [http://localhost:8000](http://localhost:8000).
